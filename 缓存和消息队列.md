### memcached
纯内存缓存，高性能分布式内存缓存，用于减轻动态web对数据库的访问。天生支持集群

启动memcached
```python
memcached -u root -m 16m -c 8292 -d -p 11211

参数：
-u指定用户，默认root
-m指定内存大小，默认64MB
-c指定最大连接数，默认1024
-P指定pid
-p指定TCP端口，默认11211
-U指定UDP端口，默认11211
-d启动守护进程
-i打印memcached和libevent证书
```
##### python-memcached客户端

第一次使用
```python
ms=memcache.Client(["10.0.0.20:11212"])
ms.set("num","dd")
v=ms.get("num")
print(v)
```

### rabbitmq
message queue消息队列。
```
#生产者
import pika
connection = pika.BlockingConnection(pika.ConnectionParameters(host="10.0.0.20"))
channel=connection.channel()
# 创建队列，指定队列名
channel.queue_declare(queue='q1')

channel.basic_publish(
    exchange="",
    routing_key="q1",
    body="shabi"
)

#消费者
connection=pika.BlockingConnection(pika.ConnectionParameters(host="10.0.0.20"))
channel=connection.channel()

def  callback(ch,method,properties,body):
    print(body)
    channel.stop_consuming()

channel.basic_consume(
    consumer_callback = callback,
    queue="q1",
    no_ack=True
)
#开始消费
channel.start_consuming()
```
